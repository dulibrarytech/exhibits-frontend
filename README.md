# exhibits-frontend

### Setup

#### Local environment

1. Clone repo:

2. cd into project folder

3. run 'npm i'

4. Set environment variables

See .env_sample for required fields

Create .env file in project root, add fields

5. run 'npm build'

6. run startup script: 'sh start_app.sh'.
